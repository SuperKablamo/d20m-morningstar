Features { beta }

The following features will be included in the first release of d20 Morningstar.
The feature list for subsequent versions will be driven by community feedback. Please join our Google Group to submit features you'd like to see in the game.

Player Characters

Players will choose from four level 1 Player Character Templates:

  • Human Fighter { Greatsword, Longbow }
  • Elf Wizard { Staff of Defense }
  • Halfling Rogue { Shortsword, Dagger }
  • Dwarf Cleric { Warhammer, Crossbow }

Non Player Characters

A variety of monster types will be released around the world including:

  • Kobolds
  • Orcs
  • Goblins
  • Bugbears
  • Skeletons
  • Wolves
  • Zombies
  • Young Chromatic Dragons
  • Vampires
  • Giant Spiders
  • Giants

Within each monster type their will be variety in level, powers, weapons and stats; providing thousands of possibilities in composition of monster parties.
